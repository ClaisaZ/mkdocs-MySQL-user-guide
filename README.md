# Purpose of this doc

This doc walks you through using [SQL Workbench](https://www.mysql.com/products/workbench/) at a basic level **on Windows**, you'll learn:

- How to create a connection
- Creating a schema (database)
- How to create tables
- Running queries in your database
- Viewing and editing data with Workbench interface.

Important: This is not a guide on how to write MySQL to manage databases, it is a guide on how to use Workbench's GUI interface to manage databases.

## Intended Reader

This guide is written for students new to SQL tools, especially anyone who’s never used a database GUI before and wants clear step-by-step instructions.

## Tone

We're keeping this friendly, simple, and practical, like walking a classmate through setup. 👍
